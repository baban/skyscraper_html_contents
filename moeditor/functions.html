イベントに反応する

萌ディタの強力なところの一つですが、ユーザーの操作に反応して動く機能を継ぎ足せること。というのがありますね。

詳しくは本家のイベントリファレンスを参照して欲しいのですが、今回はサンプルを一本紹介。

HTML書きなんかに便利なアイテム。ファイルを保存した瞬間に、IEが勝手にアクティブになってページをリロードしてくれます


f.onSave = function(){
  var shell = new ActiveXObject("WScript.Shell");  // オブジェクト呼び出し
  if( shell.AppActivate("Microsoft Internet Explorer") ){  // IEをアクティブにする
    shell.SendKeys( "{F5}" );  //F5ボタンでページをリロードする
  }
};

ショートカットなんかもコレで登録できますね
