アプリケーションの起動


// %desc  Windows標準のメモ帳を呼び出します
// %menu  メモ帳を呼び出し

var shell= new ActiveXObject("WScript.Shell");  // ActiveXを使って、シェルへの操作を出来るようにします
shell.Run("notepad.exe", 1, true);  //代表例として、ノートパッドを呼び出す

萌ディタは基幹部分に、マイクロソフトのJScriptを使っているという事は、ActiveXを使って、Windows内部の数々のプログラム部品を扱うことができますよ、という一例

WidnowsScriptingHostの方を覗けば、他にも色々とライブラリがあるので活用してみて下さい
