banner title型

サイト名等を表示するために使う、ページの上にぴっちり付いてくる形のタイトルです。

実際作ってみるとこんな感じでしょうか…。(IEだとフォントの指定出来ていませんが…)

タグの構成は下の通りです。


<h1>淀み滲む夕日の、その向こう側へ</h1>

<div>
	(このdivの中に文章を入れていきます)
</div>

スタイルの方はこんな感じ。


body {
	margin:0;
	padding:0;
	background-color:#ddf;
}
h1 {
	margin:0;
	padding:0.1em 0.5em 0.5em 0.3em;
	border:none 2px #449;
	border-bottom-style:solid;
	background-color:#76c;
	font-family:"Times New Roman", "New York", "ＤＦ平成明朝体W3", "細明朝体" , "ＭＳ 明朝", serif;
}
ポイントはマージンの指定ですが 

body {
	margin:0;
	padding:0;
}
h1 {
	margin:0;
}

こうして画面両端にピッチリくっつくようにしてから必要な文章を、divタグの中に書き込んでいきます。

そうして、h1の次に置くdivタグで、文章をインデントして、テキストを適当な位置まで戻します。
