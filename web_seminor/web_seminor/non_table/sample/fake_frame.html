position:fixed;を利用した、疑似フレームページ。

長文が、書いてあるスクロール出来ますか？

「え〜、毎回ページ書き込むの面倒くさいしさ〜」という方は外部にフレームの様にページを読み込むために<object data="ファイル名.html" type="text/html">と書き込んでやれば外部のページを読み込んでくれますね

本当は「IEではobject動かないからiframe使いましょうね〜」とか書くつもりだったのですが、IEのposition:fixed;周りの馬鹿仕様のおかげで、さっぱり動かない状況に持って行かれまして…、結局Opera,Mozilla系列専門に…

ポイントはここらへん


body {
	margin:5.5em 0 3em 0;
}
object#menu,
object#counter {
	position:fixed;
	width:100%;
	margin:0;
	left:0;
	padding:0;
}
object#menu {
	background-color:#000;
	top:0;
	height:4.5em;
}
object#counter {
	bottom:0;
	height:2em;
}

objectタグで外部のhtmlファイルを読み込んで、画面の一番上と、一番下に移動させていますね

さらにbodyタグで上と下にあるフレーム領域分だけマージンを作って、文章が全て表示されるように調整しています
